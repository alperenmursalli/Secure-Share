Private, authenticated file sharing you can host yourself.

SecShare gives every user a secure, personal space to upload and manage files over a clean REST API. Your files stay yours. Access is protected by JWT authentication.

  • ✓ Per-user isolation. You can only see, download, and delete your own files.
  • ✓ Token-based access. Sign in once and every request is authorized with a JWT.
  • ✓ Broad format support. PDF, images, text, Office documents and ZIP archives up to 50 MB.
